Descripcion del Juego: 
El juego consiste en sortear una cantidad de Amigos . Se ingresa los nombres de amigos que quisieramos sortear , no hay un limite de la cantidad de ingreso.
Una vez que empezamos a sortear, el amigo que salio sorteado, saldra de la lista para hacer el proximo sorteo. Una vez que se terminen de sortear todos los nombres. AÃ±adi un boton para que el juego se reinicie y empiece desde cero.
